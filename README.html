Shopping List API Documentation

This API allows you to manage a shopping list with basic CRUD operations.

The database is reset when the server shuts down.

Endpoints

Read All Items

GET /shoppingListItems

Returns all items in the shopping list.

Delete All Items

DELETE /shoppingListItems

Deletes all items from the shopping list.

Create Item

POST /shoppingListItem

Creates a new item in the shopping list. Requires a JSON body with 'name' (string) and 'count' (number).

Read Item

GET /shoppingListItem/:id

Returns the item with the specified ID.

Update Item

PUT /shoppingListItem/:id

Updates the item with the specified ID. Accepts a partial JSON body with 'name' (string) and/or 'count' (number).

Delete Item

DELETE /shoppingListItem/:id

Deletes the item with the specified ID.
